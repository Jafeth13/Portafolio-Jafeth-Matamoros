Smart Parking CR

Smart parking es un proyecto que se basa en la reserva de un espacio en un parqueo desde la comodidad de la casa
o donde el cliente guste mediante la página web.

Tecnologias utilizadas

Smart Parking tiene dos versiones, una todo en .NET La segunda version se desarollo el front-end con el framework de angular.Y en el backend se utilizo 2 APIs una en .NET y la otra con Spring boot. En ambas ocasiones para la base de datos se utilizo el gestor SQL Server Management Studio.
